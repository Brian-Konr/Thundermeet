# Thundermeet

## Frontend

### How to run

#### Frontend
1. Clone this repo
2. `cd` frontend
3. `yarn`
4. `yarn dev`

#### Backend
1. `cd` backend
2. `go mod download`
3. `go run main.go`

##### Appendix
- to add dependency to backend: `go get <module url>`
- to remove tidy dependency from go.mod: `go mod tidy`
